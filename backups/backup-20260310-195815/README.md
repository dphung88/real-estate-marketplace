# Prime Listings — Real Estate & Used Items Marketplace

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

A clean, responsive multi-page website for buying, selling, and renting **real estate** and **used items** (cars, lawn mowers, mobile homes, and more).

## Live Demo

[View Live Site](https://dphung88.github.io/real-estate-marketplace/)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage — hero banner, category cards, featured listings, why choose us |
| `real-estate.html` | Real estate listings — filter by type, location, price |
| `used-items.html` | Used items listings — filter by category, price, condition |
| `contact.html` | Contact form + business info + social links |

## Features

- Responsive design (mobile-friendly)
- Sticky navbar with active page highlight
- Hero section with CTA buttons
- Category grid (6 categories)
- Listing cards with badges (For Sale / For Rent / Used Item)
- Filter bar on listings pages
- Contact form with subject dropdown
- Footer with links
- Font Awesome icons
- Google Fonts (Inter)

## Project Structure

```
real-estate-marketplace/
├── index.html        # Homepage
├── real-estate.html  # Real estate listings
├── used-items.html   # Used items listings
├── contact.html      # Contact page
└── style.css         # All styles
```
