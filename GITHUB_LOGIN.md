# 🔐 Đăng nhập GitHub trong Cursor

## Cách 1: Command Palette (nhanh nhất)

1. Nhấn **`⌘ + Shift + P`** (Command + Shift + P)
2. Gõ: **`GitHub Sign`**
3. Chọn **"GitHub: Sign In"** hoặc **"GitHub: Sign in to github.com"**
4. Trình duyệt mở → đăng nhập GitHub → chọn **Authorize** để kết nối
5. Quay lại Cursor, đăng nhập hoàn tất

---

## Cách 2: Link trực tiếp

Mở link sau trong trình duyệt để kết nối tài khoản:

👉 **https://cursor.com/github-connected**

---

## Cách 3: Qua Settings

1. Nhấn **`⌘ + ,`** (mở Settings)
2. Gõ **`github`** vào ô tìm kiếm
3. Tìm **Accounts** → bấm **Sign in with GitHub**
